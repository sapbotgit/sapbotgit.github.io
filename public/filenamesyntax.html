Tags

TAG DESCRIPTION
PAL Used for games that are forced to use PAL instead of NTSC.
NTSC Used for games that are forced to use NTSC instead of PAL.
Country Version of game made for some country. Can be 'USA', 'Japan', 'Europe', 'Russia', 'World' and etc.
Unlicensed Made without licensing of console owners. Common for Russian famiclones like Dendy.
Cracked Games with cut off security. Mainly PC, almost never for consoles.
Patched Games that are not in original form (hacked/patched/modded). Common with "Cracked" games.
?? When game is not in English language. Follows ISO-639-1 standart.
Demo Not a real game/app. Just a demo.
Homebrew Made not in real studio, just in home, mostly public domain.
Public Public domain games. You can do to them whatever you want.
Prototype Unreleased version of the game. There are unreleased games too.
Mature A game that are not suitable for children. Each game has specific reasons to be there.
Setup Not installed game, not ready-to-play game. You need to install it by hand (sometimes even using cd drive emulator).
Favorite Game has been played by me, and I liked it.
VR App requires Virtual Reality headset. Not set for VR-Only platforms (like "Bjorn LOVR" or "Meta QuestOS" or "PICO OS").
Kiosk App used in kiosk hardware and never been introduced to public. Doesn't apply to kiosk-only hardware like "Arcade"
